# SQL Queries

## Overview

This folder contains the SQL scripts used in the **MedIntel AI – Hospital Management Analytics** project.

The queries support data analysis and KPI generation for multiple hospital modules.

## Modules Covered

- Patient Analytics
- Doctor Performance
- Finance Analytics
- Emergency Analytics
- Pharmacy Analytics
- Insurance Analytics
- Executive Dashboard

## SQL Concepts Used

- SELECT Statements
- INNER JOIN, LEFT JOIN
- GROUP BY
- ORDER BY
- WHERE and HAVING
- Aggregate Functions (SUM, COUNT, AVG, MAX, MIN)
- CASE Statements
- Common Table Expressions (CTEs)
- Window Functions (where applicable)

## Purpose

These SQL queries were used to:

- Analyze patient admissions
- Measure doctor performance
- Calculate revenue and billing KPIs
- Monitor emergency visits
- Track pharmacy inventory
- Analyze insurance claims
- Prepare datasets for Power BI dashboards
