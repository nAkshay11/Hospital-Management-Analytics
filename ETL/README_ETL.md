# ETL Process

## Overview
This folder contains the ETL (Extract, Transform, Load) process used in the MedIntel AI – Hospital Management Analytics project.

## Extract
- Generated realistic hospital data using Python.

## Transform
- Cleaned, validated, and formatted the data.
- Maintained relationships between tables.

## Load
- Loaded data into PostgreSQL.
- Connected PostgreSQL with Power BI for analytics.

## ETL Workflow
Python → Data Cleaning → PostgreSQL → SQL Analysis → Power BI Dashboards
